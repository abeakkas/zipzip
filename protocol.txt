Server -> Client
Initialization, sends game state and assigns id, color etc: 
{
  type: "init",
  player: {id, x, y, color},
}
New Player, adds new player to the game:
{
  type: "newplayer",
  player: {id, x, y, color}
}
Update, updates game state periodically:
{
  type: "update",
  balls: [
    {x, y, vx, vy, r},
    ...
  ],
  players: {
    id: {x, y},
    ...
  }
}

Client -> Server
Action:
{
  action: "jump" OR "left" OR "right" OR "stop"
}

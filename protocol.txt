Server -> Client
Initialization, sends game state and assigns Id: 
{
  type: "init",
  yourId: Int,
}
New Player, adds new player to the game:
{
  type: "newplayer",
  player: {id, x, y, color}
}
Update, updates game state periodically:
{
  type: "update",
  game: {
    ball: {x, y, vx, vy, r},
    players: [
      {id, x, y, color},
      ...
    ]
  }
}
